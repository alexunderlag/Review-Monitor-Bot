# Review Monitor Bot

Telegram-бот для автоматического мониторинга отзывов из **2ГИС** и **Яндекс Карт** с отправкой новых и обновлённых отзывов в Telegram-канал.

## Что делает бот
Бот регулярно проверяет отзывы у выбранных организаций в 2ГИС и Яндекс Картах, сохраняет их в локальную базу данных и отправляет в Telegram-канал только те отзывы, которые ещё не были отправлены.

Кроме новых отзывов, бот умеет отслеживать **изменения уже существующих отзывов**:
- изменение текста отзыва;
- изменение ответа компании;
- изменение рейтинга.

Если отзыв был отредактирован, бот повторно отправляет его в канал и помечает как обновлённый.

## Как работает
1. Бот запускается и инициализирует локальную базу данных SQLite.
2. Выполняется проверка отзывов из:
   - **2ГИС** через API;
   - **Яндекс Карт** через парсинг страницы отзывов.
3. Все найденные отзывы сохраняются в базу.
4. Новые и изменённые отзывы помечаются как неотправленные.
5. Бот отправляет их в Telegram-канал.
6. После успешной отправки отзыв помечается как отправленный.
7. Проверка повторяется каждые 5 минут.

## Основные возможности
- мониторинг отзывов из 2ГИС;
- мониторинг отзывов из Яндекс Карт;
- отправка отзывов в Telegram-канал;
- хранение данных в SQLite;
- отслеживание редактирования отзывов;
- повторная отправка обновлённых отзывов;
- форматирование даты, рейтинга и текста;
- обработка ограничений Telegram Flood Control;
- автоматическая миграция базы при добавлении новых полей.

## Хранение данных
Бот использует локальную базу **SQLite** и создаёт таблицы:
- `gis_reviews` — отзывы из 2ГИС;
- `yandex_reviews` — отзывы из Яндекс Карт.

Для каждого отзыва сохраняются:
- уникальный ID;
- адрес / организация;
- дата;
- автор;
- рейтинг;
- текст отзыва;
- ответ компании;
- дата последнего обновления;
- количество редактирований;
- статус отправки в Telegram.

## Источники данных
### 2ГИС
Для 2ГИС бот получает отзывы по API, используя идентификатор филиала.

### Яндекс Карты
Для Яндекс Карт бот использует Selenium и парсинг HTML-страницы отзывов.

## Назначение проекта
Проект создан для автоматизации контроля репутации компании и оперативного получения новых отзывов в Telegram-канале.

## Примечание
Проект является рабочим/учебным примером автоматизированного мониторинга отзывов и может быть доработан под другие площадки и сценарии уведомлений.
